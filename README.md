# Prime-Logic-Function
A high-accuracy prime number detection algorithm developed using functional programming principles and for loops in Python.
## Teknik Özellikler
- **Fonksiyonel Yapı:** `def asal_mı` fonksiyonu ile mantıksal işlemin ana akıştan ayrıştırılması ve kodun modüler hale getirilmesi.
- **Döngü Optimizasyonu:** `for i in range(2, sayı)` yapısı ile sayının tam bölenlerinin sistematik olarak taranması.
- **Kesintisiz İşlem:** `while True` döngüsü ile çoklu sorgulama imkanı ve güvenli çıkış (`break`) mekanizması.

## Teknik Detaylar
- **Dil:** Python 3.x
- **Kullanılan Yapılar:**
  - `Return` ifadeleri ile değer döndürme.
  - Modulo (`%`) operatörü ile tam bölünebilirlik kontrolü.
  - Tip dönüşümü (String to Int conversion).

## Nasıl Çalıştırılır?
1. Bilgisayarınızda Python yüklü olduğundan emin olun.
2. `asal_sayi_kontrol.py` dosyasını terminal üzerinden çalıştırın.
3. Kontrol etmek istediğiniz sayıyı girin; çıkmak için 'q' tuşuna basın.
